# REACT-REDUX-TOOLKIT-THUNK-FLIGHT-RADAR

<h4>

✈️ Uçuş Takip Projesi ✈️
Bu proje, özellikle Türkiye hava sahasındaki uçuşlar için gerçek zamanlı uçuş verilerini izlemeye odaklanmaktadır. Bir API aracılığıyla 300'e kadar uçuş kaydını alır ve her uçuş ve uçak hakkında ayrıntılı bilgiler sağlar.
🚀 Kullanılan Teknolojiler ve Kütüphaneler
🗃️ Reduxjs/Toolkit Uygulama durumunu verimli bir şekilde yönetmek ve güncellemek için durum yönetimi kütüphanesi.
🔗 React-Redux: Uygulama genelindeki durumu yönetmek için React bileşenlerini Redux'a bağlar.
🧭 React-Router-Dom: Farklı sayfalar veya görünümler arasında yönlendirme ve gezinme için.
✨ React-Icons: Simgelerin sorunsuz entegrasyonu için.
🎨 Bootstrap: Duyarlı ve mobil öncelikli tasarımlar için bir CSS çerçevesi.
📊 React-Paginate: Uçuş listesinde sayfalandırılmış görünümler uygulamak için.
📽 Leaflet: React Leaflet ile birlikte kullanılan, haritalar için hafif bir JavaScript kütüphanesi.
🚀 React-Leaflet: Etkileşimli haritalar oluşturmak ve uçak konumlarını görüntülemek için.
🌐 Axios: Gerçek zamanlı uçuş verilerini almak için API istekleri yapmak için.
⚡ Rapıd API: API çağrıları ve sunucu ile veri alışverişi için.
🔒 env - API anahtar bilgilerini gizlemek için kullanılır.
✈️ Uygulama Özellikleri:
🛫 Uçuş Detayları:: Rota, kalkış saati, varış saati ve tahmini süreleri görüntüler.

✈️ Uçak Detayları: Model, kuyruk kodu, ülke kodu ve uçağın bir fotoğrafını içerir.

📋 Uçuş Listesi Görünümü: Tüm aktif uçuşların sayfalandırılmış bir liste görünümü.

🔒 Güvenlik Özellikleri:
API anahtarlarını gizli ve güvenli tutmak için ortam değişkenleri kullanılarak API güvenliği sağlandı.

</h4>
